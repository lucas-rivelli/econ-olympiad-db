\question{
  id        = {0003},
  title     = {Some Rubber Duckies},
  source    = {OBECON},
  year      = {2026},
  round     = {Seletiva IEO -- Phase A},
  language  = {English},
  topic     = {Micro},
  subtopic  = {Supply \& Demand},
  type      = {objective},
  statement = {%
    The toy store ``Duck Duckies'' sells rubber duckies. The supply of rubber
    duckies is given by the inverse function $P = \dfrac{Q_s - 20}{3}$, where
    $P$ is the price in reais and $Q_s$ is the quantity supplied. The
    equilibrium price is R\$~16 and the equilibrium quantity is 68 rubber
    duckies.

    Which of the following could be the true demand curve for this market?
  },
  choices   = {%
    \item $P = 50 - 0.5Q_d$
    \item $P = 45 - 0.4Q_d$
    \item $P = 60 - 0.8Q_d$
    \item $P = 40 - 0.3Q_d$
    \item $P = 55 - 0.6Q_d$
  },
  answer    = {A},
  solution  = {%
    At equilibrium, the demand curve must satisfy $P^* = 16$ and $Q^* = 68$.
    Check each option by substituting $Q_d = 68$:
    \begin{itemize}
      \item (A) $P = 50 - 0.5 \times 68 = 50 - 34 = \mathbf{16}$ \checkmark
      \item (B) $P = 45 - 0.4 \times 68 = 17.8$ \texttimes
      \item (C) $P = 60 - 0.8 \times 68 = 5.6$ \texttimes
      \item (D) $P = 40 - 0.3 \times 68 = 19.6$ \texttimes
      \item (E) $P = 55 - 0.6 \times 68 = 14.2$ \texttimes
    \end{itemize}
    Only option \textbf{(A)} passes through the equilibrium point $(68, 16)$.
  },
}
