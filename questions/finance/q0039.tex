  type      = {objective},
  statement = {%
    QuietFork Foods compares two financing options with identical operating
    profit. Option~A uses all equity; Option~B uses a mix of debt and equity.
    Both face corporate income tax. What is the primary tax advantage of
    Option~B?
  },
  choices   = {%
    \item Interest expense reduces taxable income, lowering total taxes paid.
    \item Debt financing eliminates the need to pay taxes entirely.
    \item Operating profit is higher with debt financing.
    \item Equity holders receive tax deductions on dividends.
  },
  answer    = {A},
  solution  = {%
    Interest payments on debt are tax-deductible before calculating corporate
    taxes, whereas dividends are paid from after-tax income. This debt tax
    shield (Interest $\times$ Tax Rate) gives Option~B a lower effective tax
    burden, increasing after-tax value available to all capital providers.
    (A) is correct.
  },
}
