  statement = {%
    ObeCorp evaluates four mutually exclusive projects with similar risk.
    Alpha: NPV = \$15M, IRR = 18\%, Payback = 4~yr.
    Beta: NPV = \$12M, IRR = 22\%, Payback = 2~yr.
    Gamma: NPV = \$10M, IRR = 30\%, Payback = 1.5~yr.
    Delta: NPV = \$8M, IRR = 25\%, Payback = 1~yr.
    The cost of capital is 12\%. Which project should ObeCorp select?
  },
  choices   = {%
    \item Project Alpha, because it has the highest NPV.
    \item Project Beta, because it balances IRR and payback effectively.
    \item Project Gamma, because it has the highest IRR.
    \item Project Delta, because it has the shortest payback period.
  },
  answer    = {A},
  solution  = {%
    NPV is the correct criterion for mutually exclusive projects: it measures
    absolute value creation, accounts for the time value of money at the
    appropriate discount rate, and is consistent with shareholder wealth
    maximisation. Project Alpha's \$15M NPV exceeds all alternatives. (A) is
    correct.
  },
